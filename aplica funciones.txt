TODOS LOS CAMPOS (incluidos porcentajes, monedas, tablas y campos con listas desplejables)

- Navegacion con flechas, Tab y Enter. Si el campo tiene todo el texto seleccionado, al apretar una flecha debe pasar al campo que corresponda normalmente. Si no tiene todo el texto seleccionado, la flecha debe mover el indicador de texto dentro del mismo campo cuando corresponda.
- Seleccion de campos: cuando un campo recibe foco por primera vez, se selecciona completo. Si el campo ya esta seleccionado completo y se hace clic sobre ese mismo campo, debe deseleccionarse y colocar la barra de texto exactamente donde se hizo clic. Mientras se sigan haciendo clics sobre ese mismo campo en modo edicion, no debe volver a seleccionarse todo. Solo debe volver a seleccionarse completo al cambiar de campo y volver a recibir foco.
- F2 para seleccionar campo completo.
- Campos moneda y porcentaje: aplicar formato moneda/porcentaje en tiempo real, siempre compatible con decimales, y que el decimal del teclado numerico siempre sea tomado como decimal.
- Campos de fecha: si no se especifica mes o año se debe suponer el actual. Es decir, si hoy es 05/07/2026 y pongo 3, se autocompleta como 03/07/2026. Si pongo 2 5, se autocompleta como 02/05/2026.
- Campos Numericos: aplicar punto indicador de miles en tiempo real
- Los espacios se interpretan como separador de dia, mes y anio.
- Campos checkbox: Enter o Tab saltan al siguiente campo. Espacio marca o desmarca el check.
- Campos con listas desplegables: deben poder escribirse como texto, filtrar opciones mientras se escribe, abrir la lista con F4, navegar con flechas y seleccionar con Enter. Si el campo esta vacio y se aprieta Enter, no debe seleccionar nada automaticamente.
- Selección de campos: si el campo esta seleccionado completamente y hago 1 el campo debe desseleccionarse y colorarse la barra de texto donde se hizo click con el mouse
